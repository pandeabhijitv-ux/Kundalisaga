# Assets Folder

## Add Ganesh Image

To display the Ganesh image in the sidebar:

1. Save your Ganesh image (the one from Image 2) as `ganesh.jpg` in this `assets` folder
2. The image will automatically appear at the top of the sidebar
3. Supported formats: JPG, PNG, JPEG

**File path should be:** `C:\AstroKnowledge\assets\ganesh.jpg`

If the image is not found, a Ganesh emoji symbol will be displayed as a fallback.
